Що таке GNU/Linux

GNU/Linux — це операційна система, що складається з ядра Linux та набору утиліт GNU і додаткового програмного забезпечення. Вона широко використовується як на серверах, так і на персональних комп'ютерах. Відмінні риси: відкритий код, гнучкість налаштувань, можливість модифікації та сильна спільнота розробників.

Найпопулярніші дистрибутиви

Короткий перелік дистрибутивів, які часто використовують:

Ubuntu
Зручний для початківців, великий репозиторій
Debian
Стабільний, підходить для серверів
Fedora
Свіжі пакети, спонсор Red Hat
Arch Linux
Для просунутих користувачів, роллінг-реліз
CentOS Stream / RHEL
Для корпоративних серверів
Linux Mint
Зручний інтерфейс для колишніх користувачів Windows

Приклади базових команд CLI

Ці команди допоможуть працювати з файловою системою та базовим введенням/виведенням:

pwd
Показати поточний робочий каталог
echo "текст"
Вивести текст у консоль
ls -la
Показати список файлів з детальною інформацією
cd /шлях/до/каталогу
Перейти у вказаний каталог
touch file.txt
Створити порожній файл або оновити час модифікації
mkdir newdir
Створити нову папку
cp src dest
Копіювати файл або каталог (-r для рекурсії)
mv old new
Перемістити або перейменувати
rm file
Видалити файл (rm -r для каталогів)
history
Показати історію введених команд

BASH-скрипт — імітація авторизації (текстовий режим)

Нижче наведено приклад простого демонстраційного shell-скрипта, який імітує авторизацію в текстовому режимі. Увага: в реальних умовах не зберігайте паролі у відкритому вигляді та використовуйте механізми хешування.

# auth_demo.sh
#!/usr/bin/env bash

# Демонстраційний скрипт авторизації
# Зробіть файл виконуваним: chmod +x auth_demo.sh

declare -A USERS
USERS[alice]=password123
USERS[bob]=qwerty

read -p "Логін: " LOGIN
read -s -p "Пароль: " PASS
echo

sleep 0.4

if [[ -v USERS[$LOGIN] && "${USERS[$LOGIN]}" == "$PASS" ]]; then
  echo "Авторизація успішна. Ласкаво просимо, $LOGIN!"
  echo "$(date) - Користувач $LOGIN увійшов(ла)"
  exit 0
else
  echo "Невірний логін або пароль. Спробуйте ще раз."
  sleep 0.4
  exit 1
fi
